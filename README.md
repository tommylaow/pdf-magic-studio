# PDF Magic Studio (React & FastAPI Fullstack App)

แอปพลิเคชันเว็บระดับพรีเมียมและสวยงามที่ให้คุณจัดการไฟล์ PDF แบบลากวางเพื่อรวมและสลับหน้า (Drag & Drop) ได้อย่างลื่นไหล 100% ทำงานแบบ **Stateless (ปลอดภัยไร้รัฐ)** ไม่มีข้อจำกัดเรื่องขนาดไฟล์หรือความเร็วระบบ

## 🌟 จุดเด่นของสถาปัตยกรรมใหม่
1. **ลากวางสลับหน้าจริง**: แสดงภาพพรีวิวทุกหน้าของเอกสาร สามารถใช้เมาส์ลากย้ายสลับที่กันได้อย่างลื่นไหลและนุ่มนวลสูงสุด
2. **รันง่ายที่สุด**: รันคำสั่งเดียว สามารถเปิดระบบพร้อมกันได้ทั้งหน้าบ้าน (React) และหลังบ้าน (Python) บนพอร์ตเดียวกัน
3. **ดีไซน์สวยงาม**: พัฒนาด้วย TailwindCSS สีสันสะอาดตา ทันสมัย และตอบสนองทุกอุปกรณ์ (Responsive Design)

---

## 🚀 วิธีการรันใช้งานในเครื่องคอมพิวเตอร์ของคุณ (Local Run)

1. **เปิด Command Prompt หรือ PowerShell ในโฟลเดอร์นี้**
2. **ติดตั้งไลบรารีทั้งหมด**:
   ```bash
   pip install -r requirements.txt
   ```
3. **สั่งรันระบบ**:
   ```bash
   python main.py
   ```
4. **เปิดเว็บเบราว์เซอร์ของคุณแล้วเข้าไปที่**:
   `http://localhost:8000`

## 🌐 วิธีนำขึ้นระบบออนไลน์บน Render.com (ฟรี 100% ไม่มีค่าใช้จ่าย)

คุณสามารถนำโปรเจกต์นี้ขึ้นรันบนระบบอินเทอร์เน็ตเพื่อให้ใช้งานออนไลน์ผ่านมือถือหรืออุปกรณ์อื่นได้ตลอดเวลาตามขั้นตอนง่ายๆ ดังนี้ครับ:

### ขั้นตอนที่ 1: อัปโหลดโฟลเดอร์นี้ขึ้น GitHub
1. ทำการสร้าง Repository ใหม่บน [GitHub](https://github.com/) ของคุณ (ตั้งชื่อเรโพ เช่น `pdf-magic-studio`)
2. ในโฟลเดอร์หลักนี้ ให้สั่ง Commit และ Push โค้ดทั้งหมดขึ้น GitHub (ใช้วิธี drag-and-drop ลากไฟล์และโฟลเดอร์ `backend`, `frontend`, `main.py`, `requirements.txt` วางบนเว็บ GitHub ได้โดยตรงเลยครับ)

### ขั้นตอนที่ 2: ตั้งค่ารันฟรีบน Render.com
1. สมัครใช้งานและเข้าสู่ระบบที่เว็บไซต์ [Render.com](https://render.com/) (เชื่อมต่อด้วยบัญชี GitHub ของคุณ)
2. บนหน้าแดชบอร์ด คลิกปุ่ม **"New +"** แล้วเลือก **"Web Service"**
3. เลือก **GitHub Repository** ของคุณที่เพิ่งอัปโหลดไฟล์แอปขึ้นไป
4. ตั้งค่าหน้าเว็บดังนี้:
   * **Name**: ตั้งชื่อเว็บของคุณ (เช่น `my-pdf-studio`)
   * **Runtime**: เลือก **Python**
   * **Build Command**: `pip install -r requirements.txt`
   * **Start Command**: `python main.py`
5. เลื่อนลงมาล่างสุด เลือกแพ็คเกจแบบ **"Free"** (ฟรีตลอดชีพ) แล้วกดปุ่ม **"Create Web Service"**
6. รอระบบจัดแจง Environment และเปิดระบบประมาณ 2-3 นาที คุณจะได้ลิงก์เว็บไซต์ใช้งาน เช่น `https://my-pdf-studio.onrender.com` ที่ใช้งานออนไลน์ได้ทันทีตลอด 24 ชั่วโมงครับ!
